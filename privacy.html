Privacy Policy

Last Updated: April 29, 2026

TL;DR: PassForge doesn't collect, store, or transmit ANY of your data. Everything happens in your browser. We're 100% privacy-focused.

1. Information We Don't Collect

PassForge is designed with privacy as the foundation. We do NOT collect:

  • Generated passwords (they never leave your device)
  • Personal information (name, email, phone)
  • IP addresses or location data
  • Browser fingerprints
  • Usage statistics or analytics
  • Cookies for tracking purposes

2. How PassForge Works

PassForge operates entirely in your web browser using client-side JavaScript:

  • Local Processing: All password generation happens on your device using the Web Crypto API
  • No Server Communication: Generated passwords are never sent to our servers or any third party
  • Cryptographic Security: We use crypto.getRandomValues() for true randomness
  • Instant Generation: Passwords are created and displayed immediately without any network requests

3. Third-Party Services

PassForge uses minimal third-party services:

  • Google Fonts: Used for typography. Google may collect basic usage data. Google Privacy Policy
  • CDN Hosting: Static files may be served through a CDN for performance

We do NOT use:

  • Google Analytics or any analytics platform
  • Facebook Pixel or social media trackers
  • Advertising networks (beyond respectful, non-tracking ads)

4. Data Storage

PassForge may use browser localStorage only for:

  • Saving your preferred settings (length, character types)
  • Theme preferences (if dark/light mode is added)

This data:

  • Stays on your device only
  • Is never transmitted anywhere
  • Can be cleared at any time through browser settings

5. Security

We implement security best practices:

  • HTTPS Only: All traffic is encrypted
  • No Backend: Static site architecture eliminates server-side vulnerabilities
  • Content Security Policy: Protects against XSS attacks
  • Subresource Integrity: Ensures external resources aren't tampered with

6. Your Rights

Since we don't collect personal data, there's nothing to:

  • Access (we don't have your data)
  • Delete (we don't store your data)
  • Export (we don't collect your data)
  • Correct (we don't maintain your data)

7. Children's Privacy

PassForge can be used by anyone. Since we don't collect any personal information, we don't have age restrictions. However, we recommend parental guidance for minors using online security tools.

8. Changes to This Policy

We may update this privacy policy occasionally. Changes will be posted on this page with an updated "Last Updated" date. Since we don't collect contact information, we can't notify you directly of changes.

9. Open Source

PassForge's code is transparent and can be inspected. You can verify that we don't collect or transmit data by reviewing the source code or using browser developer tools to monitor network activity.
